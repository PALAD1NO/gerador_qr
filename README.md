# Gerador de QR Code Pro - Versão 3.1 🚀

Este aplicativo foi desenvolvido em **Flutter** para atender a uma demanda interna específica, otimizando o fluxo de trabalho e a geração de identificadores rápidos na minha empresa atual.

## 📋 O Problema & A Solução
Antes, a geração de etiquetas ou códigos seguia um processo manual ou dependia de ferramentas genéricas lentas. Este app foi projetado para ser um **totem de produtividade**, permitindo que qualquer colaborador gere códigos padronizados em segundos.

## ✨ Funcionalidades Principais
- **Entrada Padronizada:** Campos de texto limitados a 2 dígitos para evitar erros de digitação humanos.
- **Atalhos Rápidos:** Botões dedicados para os códigos mais utilizados na operação (A5-EXP, A6-EXP, A7-EXP, A8-EXP).
- **Histórico Inteligente:** Grade visual com os últimos 10 códigos gerados para reutilização imediata.
- **Trava de Orientação:** Interface fixada no modo vertical para facilitar o manuseio com apenas uma mão em ambiente de estoque/produção.
- **Compartilhamento Direto:** Opção de enviar a imagem do QR Code instantaneamente via sistemas nativos do Android.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Dart
- **Framework:** [Flutter](https://flutter.dev)
- **Persistência:** SharedPreferences (Armazenamento local do histórico)
- **Bibliotecas:** qr_flutter, screenshot, share_plus

## 📱 Interface
A interface foi ajustada para máxima legibilidade (Versão 3.1), com fontes ampliadas e botões de toque fácil, garantindo eficiência mesmo em aparelhos com diferentes configurações de escala.
